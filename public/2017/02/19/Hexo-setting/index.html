Hexo 配置与常用命令

hexo常用命令

  • 初始化
    hexo init dirname

  • 新建文章
    hexo n "myblog" == hexo new "myblog"

  • 新建草稿
    hexo new draft " new_draft"
    草稿在使用hexo g的时候并不发布

  • 发布草稿
    hexo publish "new_draft"
  • 生成网页
    hexo g == hexo generate
  • 启动服务器预览
    hexo s == hexo server
  • 部署到git
    hexo d == hexo deploy
  • 清除缓存
    hexo clean
